# Contributing to Kerbonaut Redux

Thanks for wanting to help out! Here's how:

## Quick Rules

- Be nice
- Test your changes
- Keep PRs focused on one thing
- Don't break existing stuff

## What You Can Contribute To

This repo has three parts:

### 1. KerbonautRedux-Mod (the KSP mod)
- C# code in `KerbonautRedux-Mod/src/`
- Custom cosmetics for kerbals

### 2. KerbonautRedux-CosmeticManager (the tool)
- Python code in `KerbonautRedux-CosmeticManager/`
- GUI for managing cosmetics

### 3. CrewPortraitFix (the camera fix)
- C# code in `CrewPortraitFix/src/`
- Fixes for crew portrait camera

## How to Contribute

### Found a Bug?
- Open an issue
- Say which part (Mod, Manager, or Portrait Fix)
- Include steps to reproduce
- KSP version helps for mod bugs

### Want to Add Something?
- Open an issue first to discuss
- Or just send a PR if it's small

### Sending a PR

1. Fork the repo
2. Make a branch: `git checkout -b my-cool-feature`
3. Do your thing
4. Test it actually works
5. Submit PR

## By Contributing

You agree to license your stuff under the same license as this project (see LICENSE).

## Questions?

Email me@onge.org or open an issue.

Thanks!
